Política de Privacidad

Esta aplicación de mensajería automatizada para Instagram Business no recopila, almacena ni comparte información personal del usuario. Utiliza únicamente los permisos otorgados por el usuario para responder mensajes directos dentro del entorno de Meta (Facebook e Instagram).

No compartimos información con terceros. Toda interacción se realiza dentro del entorno seguro de la API de Meta.

Para dudas o consultas, puedes escribir a: contacto@tudominio.com
